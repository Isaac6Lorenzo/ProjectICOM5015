Para correr el programa utiliza el comando en el terminal:
    ==> python main.py

Se abrira la interface de tkinter en donde eliges:
    1. Modo Terminal: libreria de python de Sudoku vs Backtraking usando tiempo de ejecution.
    2. Modo Player: donde el player puede jugar Sudoku y si se estanca en buscar la solucion 
        solo preciona la tecla espacio or SPACE para resolver el Sudoku usando el Backtracking.
    3. Eliges la dificultad del Sudoku. Hay tres niveles de dificultad: Easy, Medium y Hard.
    4. Cuando tenga selecionado el modo y la dificultad, entonces preciona el boton enter de la
        interface de tkinter.
    5. Entonces la interface de tkinter desaparecera y entrara en accion el programa.
