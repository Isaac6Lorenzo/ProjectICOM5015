Para correr el programa usa el comando:
	python main.py

Dentro de la carpeta de CheckersMinimax

Funcionamiento:
	1. Abre la interface de Tkinter
	2. Escoge entre Player vs AI o AI vs AI.
	3. Escoge el "depth" del Minimax Search
	4. Escoge en el modo entre: 
		1) Supervivencia - procura mantener el maximo de fichas jugables posible
		2) Ataque - procura mantener un contador lo maximo posible de fichas comidas
	5. Despues de selecionar la configuracion, presionar enter
	6. El boton de on/off es para cerrar la interface de tkinker y tambien el boton X
